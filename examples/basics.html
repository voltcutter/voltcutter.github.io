How to start a Unity project

1. Before you actually make a Unity project, you need to have a version installed. If you have one installed, you are okay to go. However, if you don't, you should install one by going to "Installs" on Unity Hub, and clicking "Add". Follow instructions from there.

2. Go back to "Projects" and click "New".

3. There's probably a bunch of templates that you can use, so choose one.

4. Wait for it to load, and now you can start creating your game! If you need more examples, check the table of contents.
